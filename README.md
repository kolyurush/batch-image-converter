Пакетный конвертер изображений (Batch Image Converter)

Курсовой проект по разработке утилиты для быстрой параллельной конвертации изображений.

  Особенности
- Многопроцессорность (`multiprocessing`): Ускоренная обработка за счет использования всех ядер CPU.
- Два режима работы: Поддержка CLI (интерфейс командной строки через `argparse`) и современного GUI на `CustomTkinter`.
- Автоматическое тестирование: Покрытие логики тестами с помощью `pytest`.

  Структура проекта
- `src/` — исходный код (логика, интерфейс, точка входа).
- `tests/` — юнит-тесты.
- `requirements.txt` — фиксированные зависимости.

  Запуск проекта
1. Установка зависимостей: `pip install -r requirements.txt`
2. Запуск GUI: `python -m src.main`
3. Запуск CLI: `python -m src.main -i /путь/откуда -o /путь/куда -f WEBP`
4. Запуск тестов: `pytest`
